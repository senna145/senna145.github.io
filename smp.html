SMP Server

Algemene Informatie

Wat is SMP?
SMP staat voor Survival Multiplayer.

Op welke versie draait de server?
De server is momenteel beschikbaar voor 1.17.1.

Wat voor server is het?
Deze server is een Vanilla 1.17.1 server.

Regels

Natuurlijk heeft de server ook een aantal standaard regels.

  • Geen cheats
  • Geen hacks
  • Niet griefen
  • Niet stelen

Hoe speel ik mee?

Om mee te spelen met de server moet je in de "Max Plank" discord zitten.

Zodra je gejoined bent laat je weten dat je mee wilt spelen op de server.

Vervolgens krijg je toestemming tot de server-gerelateerde kanalen met alle informatie, waaronder het IP-adress.
